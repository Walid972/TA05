Tienda de Zapatillas (Aplicación de Escritorio)

Descripción

Este proyecto consiste en una aplicación de escritorio de una tienda de zapatillas, desarrollada en Python utilizando Tkinter. Permite visualizar productos, gestionarlos en un carrito y finalizar compras.

Los productos se cargan desde archivos JSON con información como nombre, precio e imagen. El sistema calcula automáticamente el total y guarda el registro de la compra.

Detalles Técnicos

  • Tkinter para la interfaz gráfica.
  • Pillow para imágenes.
  • JSON para almacenamiento.
