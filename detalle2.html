Tienda de Zapatillas (Aplicación de Escritorio)

Descripción

Este proyecto consiste en una aplicación de escritorio de una tienda de zapatillas, desarrollada en Python utilizando la librería Tkinter para la interfaz gráfica. La aplicación simula un sistema de compra en el que el usuario puede visualizar productos, añadirlos a un carrito, eliminar artículos y finalizar una compra.

Los productos se cargan desde un archivo JSON, que almacena información como el nombre, precio e imagen de cada zapatilla. La aplicación presenta los productos de forma visual y permite modificar la disposición de la vista para mejorar la experiencia del usuario. El sistema incluye un carrito de compras funcional que calcula automáticamente el total y guarda el registro de la compra en un archivo JSON.

Además, el proyecto incorpora un sistema básico de gestión de credenciales, que permite verificar la contraseña actual del usuario y establecer una nueva que cumpla ciertos requisitos de seguridad.

Más Detalles Técnicos

La aplicación está desarrollada con un enfoque práctico y educativo, utilizando tecnologías fundamentales de Python:

  • Tkinter para la creación de la interfaz gráfica, ventanas, botones y eventos.
  • PIL (Pillow) para la carga y redimensionado de imágenes de los productos.
  • JSON como sistema de almacenamiento de datos.

Archivos JSON utilizados:

  • zapatillas.json: información de los productos.
  • credencials.json: datos de autenticación del usuario.
  • compra.json: registro de compras realizadas.

Funcionalidades del carrito de compras:

  • Añadir y eliminar productos dinámicamente.
  • Cálculo automático del total de la compra.
  • Almacenamiento de la compra final en un archivo JSON.

No se han utilizado frameworks avanzados, lo que permite centrarse en los fundamentos de la programación, la gestión de archivos, estructuras de datos y el desarrollo de interfaces gráficas. El proyecto demuestra conocimientos en Python, programación orientada a eventos y manejo de datos, ofreciendo una experiencia clara y funcional.
