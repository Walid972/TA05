Tienda de Zapatos Online

Descripción

Este proyecto es mi primera web completa, una tienda de zapatos online desarrollada utilizando tecnologías básicas pero efectivas: HTML para la estructura, CSS para el diseño y estilos, JavaScript para la interactividad y JSON para almacenar y gestionar los datos de los productos. La tienda permite a los usuarios navegar por una colección de zapatos, ver detalles de cada producto, agregar artículos al carrito de compras y simular un proceso de compra. Incluye características como filtros por categoría, búsqueda de productos y una interfaz responsiva que se adapta a dispositivos móviles y de escritorio.

Más Detalles Técnicos

El proyecto se basa en un enfoque modular: el HTML define la estructura de las páginas (inicio, catálogo, carrito), el CSS proporciona un diseño moderno y atractivo con animaciones sutiles, JavaScript maneja la lógica del carrito (agregar, eliminar, calcular totales) y JSON se utiliza para simular una base de datos local con información de productos como nombre, precio, imagen y descripción. No se utilizaron frameworks avanzados para mantenerlo simple y educativo, enfocándome en los fundamentos del desarrollo web. El resultado es una experiencia de usuario fluida que demuestra habilidades en manipulación del DOM, eventos de JavaScript y gestión de datos.
